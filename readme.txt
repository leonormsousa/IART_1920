How to Compile:
In your console, go to the directory of file game.cpp. Run the command: g++ -o game game.cpp. For this you shoul have the c++ compiler: g++.

How to Run: Run, in the console, the command: ./game <mode> <level>
<mode> -> game modes:
   1 = human
   2 = breath_first
   3 = depth_first
   4 = iterative deepening
   5 = greedy
   6 = A*
<level> -> level: from 1 to 7
Note: level 5 does not have a solution.
